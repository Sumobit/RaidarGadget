How to deploy the gadget:


1. Open 'RaidarGadget.sln' in visual studio.

2. Choose 'Publish RaidarGadget' from the build menu

3. Click 'Finish'

4. Zip the contents of the 'Website' folder while keeping the directory structure intact

5. Rename the zipfile to 'RaidarGadget.gadget'

6. Double click 'RaidarGadget.gadget' to install.
